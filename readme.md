# Nnaemeka Academy

A single-page application built with vanilla HTML, CSS, and JavaScript. Navigation dynamically loads page content without any page reloads.

## Project Structure

```
firstjs/
├── index.html
├── app/
│   └── app.js
├── css/
│   └── styles.css
├── imgs/
└── readme.md
```

## Features

- Dynamic single-page navigation using `innerHTML`
- Active link state tracked with `classList`
- Fade-in animation on page content transitions
- Fully responsive layout
- No external libraries or frameworks
